# Landing Page Template

Reusable HTML template for fast, SEO-friendly landing pages.

## Purpose
This repository serves as a base template for creating multiple static landing pages.

## Structure
- index.html – main landing page
- assets/css/style.css – styles
- assets/img/ – images

## Usage
1. Click "Use this template"
2. Rename the repository
3. Update content in index.html
4. Deploy via Cloudflare Pages, Netlify, or Vercel

## Notes
No frameworks. Pure HTML/CSS for maximum speed and SEO control.
